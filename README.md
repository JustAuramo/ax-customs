#### Description

Basic Vehicle tuning and customize menu! (edited version for qb-customs)

---

#### Features



- Tuning Your Car
- Custom Location
- Change Livery
- Customize Price
- Easy to use and read

---

### Installation

- Add this text in your `server.cfg` :

```
ensure qb-customs
```

---

### Config

- Open the config.lua file and edit it the way you want :

```
config.lua
```
